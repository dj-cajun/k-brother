## 0. 시스템 개요

교민 커뮤니티 사이트 배너 유입을 통해 회원에게 **10% 할인 카페 스아다 쿠폰**을 발급하고, 현장에서 **3분 한시 만료 보안 QR**로 사용·소멸 처리하며, 대리점 실시간 알림 및 운영자 Discord Webhook 알림까지 처리하는 O2O MVP 시스템.

### 0.1 핵심 설계 원칙

| 원칙 | 설명 |
|------|------|
| 보안 QR | QR은 단순 ID가 아닌 **서명된(JWT) 1회성 토큰**, 3분 TTL |
| 원자적 소멸 | 쿠폰 소멸은 DB 트랜잭션 + 상태머신으로 **이중 사용 방지** |
| 실시간성 | WebSocket(Socket.IO)로 대리점 알림, Discord Webhook로 운영자 알림 |
| 멱등성 | 스캔 검증 API는 멱등 처리 (중복 스캔 무해화) |

### 0.2 기술 스택

```
Runtime    : Node.js 20 LTS
Framework  : Express 4
Realtime   : Socket.IO 4
DB         : PostgreSQL 15 (Prisma ORM 권장, 본 MVP는 pg 직접 사용)
Auth       : JWT (jsonwebtoken)
QR         : qrcode (생성), JWT 서명 토큰
Notify     : Discord Webhook (axios)
```

## 3. 상태머신 (쿠폰 라이프사이클)

```
ISSUED ──(use 요청)──▶ PENDING(QR active 3min)
   ▲                        │
   │ TTL 만료 (재사용 가능)  │ scan 검증 성공
   └────────────────────────┤
                            ▼
                        REDEEMED (소멸/최종)
```

- `ISSUED` : 지갑에 발급된 상태
- `PENDING` : QR 생성됨, 3분 타이머 작동. 만료 시 다시 `ISSUED`로 복귀 가능
- `REDEEMED` : 스캔 완료, **소멸(최종 불변)**

```markdown
| 컬럼 | 타입 | 제약 | 설명 |
|------|------|------|------|
| id | UUID | PK | 대리점 ID |
| store_name | VARCHAR(150) | NOT NULL | 매장명 |
| login_id | VARCHAR(100) | UNIQUE, NOT NULL | 로그인 ID |
| password_hash | VARCHAR(255) | NOT NULL | bcrypt 해시 |
| created_at | TIMESTAMPTZ | default now() | |

## 3. coupons — 쿠폰
| 컬럼 | 타입 | 제약 | 설명 |
|------|------|------|------|
| id | UUID | PK | 쿠폰 ID |
| user_id | UUID | FK→users.id, NOT NULL | 소유 유저 |
| code | VARCHAR(20) | UNIQUE, NOT NULL | 사람 식별용 코드 |
| title | VARCHAR(150) | NOT NULL | "카페 스아다 10% 할인" |
| discount_rate | INT | NOT NULL default 10 | 할인율(%) |
| status | VARCHAR(20) | NOT NULL default 'ISSUED' | ISSUED/PENDING/REDEEMED |
| qr_jti | VARCHAR(64) | NULL | 현재 활성 QR 토큰 jti |
| qr_expires_at | TIMESTAMPTZ | NULL | QR 만료 시각(3분) |
| redeemed_at | TIMESTAMPTZ | NULL | 소멸 시각 |
| created_at | TIMESTAMPTZ | default now() | 발급일 |

CHECK (status IN ('ISSUED','PENDING','REDEEMED'))

## 4. transactions — 트랜잭션(지급 이력)
| 컬럼 | 타입 | 제약 | 설명 |
|------|------|------|------|
| id | UUID | PK | 트랜잭션 ID |
| coupon_id | UUID | FK→coupons.id, UNIQUE | 1쿠폰 1지급 |
| agent_id | UUID | FK→agents.id | 처리 대리점 |
| user_id | UUID | FK→users.id | 유저 |
| redeemed_at | TIMESTAMPTZ | default now() | 지급 처리 시각 |
| notified_discord | BOOLEAN | default false | Webhook 발송 여부 |

## 인덱스
- coupons(user_id), coupons(status), coupons(qr_jti)
- transactions(agent_id), transactions(coupon_id UNIQUE)

## 상태 전이 무결성
- ISSUED→PENDING : QR 생성
- PENDING→REDEEMED : 스캔 검증 (UPDATE ... WHERE status='PENDING' 원자성 보장)
- PENDING→ISSUED : qr_expires_at 만료 시 재사용 허용
```

## 02-Workspace/src/db.js

```javascript
const fs = require('fs');
const path = require('path');
const { Pool } = require('pg');
const cfg = require('./config');

const pool = new Pool({ connectionString: cfg.databaseUrl });

pool.on('error', (err) => {
  console.error('[DB] Unexpected idle client error', err);
});

async function query(text, params) {
  return pool.query(text, params);
}

/**
 * 트랜잭션 헬퍼: 콜백에 client 전달, 자동 commit/rollback
 */
async function withTransaction(fn) {
  const client = await pool.connect();
  try {
    await client.query('BEGIN');
    const result = await fn(client);
    await client.query('COMMIT');
    return result;
  } catch (err) {
    await client.query('ROLLBACK');
    throw err;
  } finally {
    client.release();
  }
}

async function migrate() {
  const sql = fs.readFileSync(path.join(__dirname, '../sql/schema.sql'), 'utf8');
  await pool.query(sql);
  console.log('[DB] Migration completed.');
}

module.exports = { pool, query, withTransaction, migrate };
```

## 02-Workspace/src/realtime.js

```javascript
const { Server } = require('socket.io');

let io = null;

function init(httpServer) {
  io = new Server(httpServer, { cors: { origin: '*' } });

  io.on('connection', (socket) => {
    // 대리점 클라이언트는 접속 후 자신의 룸에 join
    socket.on('agent:join', (agentId) => {
      if (typeof agentId === 'string' && agentId.length > 0) {
        socket.join(`agent:${agentId}`);
        socket.emit('agent:joined', { room: `agent:${agentId}` });
      }
    });
  });

  return io;
}

/** 특정 대리점 룸으로 음료 지급 알림 발송 */
function emitDispense(agentId, payload) {
  if (!io) return;
  io.to(`agent:${agentId}`).emit('drink:dispense', payload);
}

module.exports = { init, emitDispense };
```

## 02-Workspace/src/middleware/auth.js

```javascript
const jwt = require('jsonwebtoken');
const cfg = require('../config');

function makeAuth(expectedRole) {
  return (req, res, next) => {
    const header = req.headers.authorization || '';
    const token = header.startsWith('Bearer ') ? header.slice(7) : null;
    if (!token) return res.status(401).json({ error: 'NO_TOKEN' });

    try {
      const payload = jwt.verify(token, cfg.jwtSecret);
      if (expectedRole && payload.role !== expectedRole) {
        return res.status(403).json({ error: 'FORBIDDEN_ROLE' });
      }
      req.auth = payload; // { sub, role, name }
      next();
    } catch {
      return res.status(401).json({ error: 'INVALID_TOKEN' });
    }
  };
}

module.exports = {
  userAuth: makeAuth('user'),
  agentAuth: makeAuth('agent'),
};
```

## 02-Workspace/src/services/qrService.js

```javascript
const jwt = require('jsonwebtoken');
const QRCode = require('qrcode');
const { v4: uuidv4 } = require('uuid');
const cfg = require('../config');

/**
 * 3분 한시 보안 QR 토큰 생성.
 * - jti: 1회성 식별자 (DB와 매칭하여 재사용 방지)
 * - exp: 3분 TTL
 */
function signQrToken({ couponId, userId }) {
  const jti = uuidv4();
  const token = jwt.sign(
    { couponId, userId, typ: 'coupon_qr' },
    cfg.qrSecret,
    { expiresIn: cfg.qrTtlSeconds, jwtid: jti }
  );
  const expiresAt = new Date(Date.now() + cfg.qrTtlSeconds * 1000);
  return { token, jti, expiresAt };
}

function verifyQrToken(token) {
  return jwt.verify(token, cfg.qrSecret); // throws on expire/invalid
}

async function toDataUrl(token) {
  return QRCode.toDataURL(token, { errorCorrectionLevel: 'M', margin: 1 });
}

module.exports = { signQrToken, verifyQrToken, toDataUrl };
```

## 02-Workspace/src/services/couponService.js

```javascript
const crypto = require('crypto');
const db = require('../db');
const qrService = require('./qrService');
const notify = require('./notifyService');

function genCouponCode() {
  return 'SUADA-' + crypto.randomBytes(3).toString('hex').toUpperCase();
}

/** 회원가입 시 환영 쿠폰 발급 */
async function issueWelcomeCoupon(client, userId) {
  const code = genCouponCode();
  const { rows } = await client.query(
    `INSERT INTO coupons (user_id, code, title, discount_rate, status)
     VALUES ($1, $2, $3, 10, 'ISSUED') RETURNING *`,
    [userId, code, '카페 스아다 10% 할인 쿠폰']
  );
  return rows[0];
}

async function listWallet(userId) {
  const { rows } = await db.query(
    `SELECT id, code, title, discount_rate, status, qr_expires_at, redeemed_at, created_at
       FROM coupons WHERE user_id = $1 ORDER BY created_at DESC`,
    [userId]
  );
  return rows;
}

/**
 * 3분 QR 생성.
 * 상태머신: ISSUED -> PENDING (또는 PENDING이며 만료된 경우 갱신)
 */
async function generateQr(userId, couponId) {
  return db.withTransaction(async (client) => {
    const { rows } = await client.query(
      `SELECT * FROM coupons WHERE id = $1 AND user_id = $2 FOR UPDATE`,
      [couponId, userId]
    );
    const coupon = rows[0];
    if (!coupon) throw httpError(404, 'COUPON_NOT_FOUND');
    if (coupon.status === 'REDEEMED') throw httpError(409, 'ALREADY_REDEEMED');

    const { token, jti, expiresAt } = qrService.signQrToken({ couponId, userId });

    await client.query(
      `UPDATE coupons SET status='PENDING', qr_jti=$1, qr_expires_at=$2 WHERE id=$3`,
      [jti, expiresAt, couponId]
    );

    const dataUrl = await qrService.toDataUrl(token);
    return { token, dataUrl, expiresAt };
  });
}

/**
 * 대리점 QR 검증 + 즉시 소멸 (원자적).
 * 멱등성: 이미 REDEEMED인 동일 쿠폰 재스캔은 충돌 처리.
 */
async function verifyAndRedeem(agentId, agentStoreName, qrToken) {
  // 1. 토큰 서명/만료 검증 (만료 시 throw)
  let decoded;
  try {
    decoded = qrService.verifyQrToken(qrToken);
  } catch (e) {
    throw httpError(400, e.name === 'TokenExpiredError' ? 'QR_EXPIRED' : 'QR_INVALID');
  }
  if (decoded.typ !== 'coupon_qr') throw httpError(400, 'QR_INVALID');

  const result = await db.withTransaction(async (client) => {
    // 2. 쿠폰 락
    const { rows } = await client.query(
      `SELECT c.*, u.name AS user_name
         FROM coupons c JOIN users u ON u.id = c.user_id
        WHERE c.id = $1 FOR UPDATE`,
      [decoded.couponId]
    );
    const coupon = rows[0];
    if (!coupon) throw httpError(404, 'COUPON_NOT_FOUND');
    if (coupon.status === 'REDEEMED') throw httpError(409, 'ALREADY_REDEEMED');
    if (coupon.status !== 'PENDING') throw httpError(409, 'QR_NOT_ACTIVE');

    // 3. jti 일치(현재 활성 QR인지) 및 만료 재검증
    if (coupon.qr_jti !== decoded.jti) throw httpError(409, 'QR_SUPERSEDED');
    if (new Date(coupon.qr_expires_at).getTime() < Date.now()) {
      throw httpError(400, 'QR_EXPIRED');
    }

    // 4. 원자적 소멸: status PENDING -> REDEEMED 조건부 업데이트
    const upd = await client.query(
      `UPDATE coupons
          SET status='REDEEMED', redeemed_at=now(), qr_jti=NULL, qr_expires_at=NULL
        WHERE id=$1 AND status='PENDING'
        RETURNING redeemed_at`,
      [coupon.id]
    );
    if (upd.rowCount === 0) throw httpError(409, 'CONCURRENT_REDEEM');

    // 5. 트랜잭션 기록
    const tx = await client.query(
      `INSERT INTO transactions (coupon_id, agent_id, user_id, redeemed_at)
       VALUES ($1, $2, $3, $4) RETURNING id`,
      [coupon.id, agentId, coupon.user_id, upd.rows[0].redeemed_at]
    );

    return {
      transactionId: tx.rows[0].id,
      couponCode: coupon.code,
      title: coupon.title,
      userName: coupon.user_name,
      redeemedAt: upd.rows[0].redeemed_at,
    };
  });

  // 6. 알림 (커밋 후, fire-and-forget)
  notify.notifyAgentRealtime(agentId, result);
  notify.notifyDiscord({ ...result, storeName: agentStoreName })
    .then((ok) => {
      if (ok) {
        db.query(`UPDATE transactions SET notified_discord=true WHERE id=$1`, [result.transactionId])
          .catch((e) => console.error('[DB] flag update failed', e.message));
      }
    });

  return result;
}

function httpError(status, code) {
  const e = new Error(code);
  e.status = status;
  e.code = code;
  return e;
}

module.exports = { issueWelcomeCoupon, listWallet, generateQr, verifyAndRedeem, httpError };
```

```javascript
const express = require('express');
const bcrypt = require('bcryptjs');
const jwt = require('jsonwebtoken');
const db = require('../db');
const cfg = require('../config');
const couponService = require('../services/couponService');

const router = express.Router();

// 회원가입 + 환영 쿠폰 자동 발급
router.post('/signup', async (req, res, next) => {
  try {
    const { email, password, name, referralSource } = req.body;
    if (!email || !password || !name) return res.status(400).json({ error: 'MISSING_FIELDS' });

    const hash = await bcrypt.hash(password, 10);

    const result = await db.withTransaction(async (client) => {
      const exists = await client.query('SELECT 1 FROM users WHERE email=$1', [email]);
      if (exists.rowCount > 0) throw couponService.httpError(409, 'EMAIL_TAKEN');

      const u = await client.query(
        `INSERT INTO users (email, password_hash, name, referral_source)
         VALUES ($1,$2,$3,$4) RETURNING id, email, name`,
        [email, hash, name, referralSource || 'gyomin_banner']
      );
      const user = u.rows[0];
      const coupon = await couponService.issueWelcomeCoupon(client, user.id);
      return { user, coupon };
    });

    const token = jwt.sign(
      { sub: result.user.id, role: 'user', name: result.user.name },
      cfg.jwtSecret, { expiresIn: '7d' }
    );
    res.status(201).json({ token, user: result.user, welcomeCoupon: result.coupon });
  } catch (e) { next(e); }
});

// 유저 로그인
router.post('/login', async (req, res, next) => {
  try {
    const { email, password } = req.body;
    const { rows } = await db.query('SELECT * FROM users WHERE email=$1', [email]);
    const user = rows[0];
    if (!user || !(await bcrypt.compare(password, user.password_hash))) {
      return res.status(401).json({ error: 'INVALID_CREDENTIALS' });
    }
    const token = jwt.sign(
      { sub: user.id, role: 'user', name: user.name },
      cfg.jwtSecret, { expiresIn: '7d' }
    );
    res.json({ token, user: { id: user.id, email: user.email, name: user.name } });
  } catch (e) { next(e); }
});

// 대리점 로그인
router.post('/agent/login', async (req, res, next) => {
  try {
    const { loginId, password } = req.body;
    const { rows } = await db.query('SELECT * FROM agents WHERE login_id=$1', [loginId]);
    const agent = rows[0];
    if (!agent || !(await bcrypt.compare(password, agent.password_hash))) {
      return res.status(401).json({ error: 'INVALID_CREDENTIALS' });
    }
    const token = jwt.sign(
      { sub: agent.id, role: 'agent', name: agent.store_name },
      cfg.jwtSecret, { expiresIn: '1d' }
    );
    res.json({ token, agent: { id: agent.id, storeName: agent.store_name } });
  } catch (e) { next(e); }
});
```

## 6. 보안·무결성 체크리스트

| 항목 | 처리 |
|------|------|
| QR 위조 방지 | JWT 서명(QR_SECRET) + jti 매칭 |
| 3분 만료 | JWT `exp` + DB `qr_expires_at` 이중 검증 |
| 이중 사용 방지 | `FOR UPDATE` 락 + 조건부 `UPDATE ... WHERE status='PENDING'` |
| 동시성 | rowCount 검사로 race condition 차단 |
| 권한 분리 | role 기반 JWT (`user` / `agent`) |
| 알림 실패 격리 | Discord 실패가 소멸 트랜잭션에 영향 없음 |
| 비밀번호 | bcrypt 해시 |
